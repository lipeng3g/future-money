# .agent (协同开发区)

此目录用于把“人类 + 多个 AI/worker”协作所需的上下文**放进仓库**，让所有协作者都可见、可审阅、可 PR。

## 规则
- ✅ 可以放：任务拆解、验收标准、设计/决策记录、工作日志、调试结论（去隐私/去密钥）、常用提示词模板
- ❌ 绝不放：token/cookie/私钥/个人账号信息、任何敏感原始日志

## 文件
- `NEXT_TASK.md`：任务真相源（worker 必须先读）
- `WORKLOG.md`：每次交付后追加一条记录（含如何验收）
- `DECISIONS.md`：关键决策沉淀
- `WORKER_PROTOCOL.md`：约束 worker 工作方式（最小可验收改动 + 可审阅）
