## 一句话概括

把每一笔（含周期性）资金变动记录为**真实可编辑的记录**，用现代化前台界面直观推演未来数月、数年的资金走势，并支持任意单笔与整组的增删改。

## 核心设计决策速查

- **真实记录原则**：周期性收支创建时即展开为一笔笔独立 `Transaction`，任意一笔（含未来）可单独改/删。
- **批量管理**：按时间列出整组、默认全选、自由勾选后批量删/改。
- **技术栈**：React + TypeScript + Vite + Semi Design + VChart + Zustand + dayjs。
- **存储**：纯前端 localStorage + JSON 导入/导出备份。
- **界面**：现代化前台（图表为中心、卡片化、浮层引导），非后台管理系统。
- **本期不做**：AI、多币种、多语言、云同步。
